Method command (Edit menu)

Use this command to open the Edit method dialog box.

You can edit the properties of already computed method(s) and/or delete method results which are not used any more. Click  Current Profile radio button to see only methods computed for current profile in active view. To see all computed methods, click the All Profiles radio button. To quickly select a method as current double click on the method in the list.

By clicking Set as current button, you select the method for further manipulation in the view. Selected method is displayed on Status Bar.  

Show/Hide button shows or hides the currently selected method in the list.

Delete button temporary removes the method from the list.

Hit Properties button to display and modify the methods properties.

Rem. hidden solution temporary deletes all hidden (grayed) solutions.

Export writes the solutions to a text file.

When you finished editing Close this dialog box.
