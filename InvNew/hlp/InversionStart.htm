Starting with inversion.

Inversion is geophysical inversion software for magnetic and gravity profile data. It is using Werner deconvolution to compute approximate position of singularities of anomalous bodies. For detailed description see Werner deconvolution. In this document you will find complex dectription abou what Inversion is, and how to work with Inversion. The picture below provides an overwiev at Inversion application.

How to start?

    1. Open an instance of Inversion either doubleclick the icon on your Desktop or from Program menu.
    2. From menu file choose New. This will open new, empty document.
    3. Import your data in menu Database / Import. This command is importing profile data in format specifiet here.
    4. From menu Method choose Werner. This will open Werner Deconvolution dialog box.
