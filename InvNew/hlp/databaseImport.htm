Import data file format.

Database data to import are text files *.xyz (extension *.xyz is not necessary). The format of the file follows:

Keywods:
    // - tag denotes comment, once this tag appears on a line, the line is ignored
    profile [number] - starts the profile data section. from the next line in file data reading starts. [number] is your profile number.

Fist data column are fiduacials. This column is inevitable. One more field data column is inevitable. Next columns are optional. After the data section can follow next profile.

Follows an example of file.

// deriv dx dz
profile 1
